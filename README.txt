# Portfolio CV (version statique)

Ce dossier contient une version **100% statique** (HTML/CSS/JS) du projet : pas de Node, pas de BDD, pas d'Express.

## Lancer en local
- Ouvre `index.html` dans ton navigateur
- (Recommandé) avec un petit serveur local :
  - VS Code: extension "Live Server"
  - ou `python3 -m http.server 8000` puis http://localhost:8000

## Où modifier
- `index.html` : ton nom, titre, liens, texte "À propos", cartes projets, contact
- `images/` : ta photo, ton CV PDF, etc.
- `stylesheets/style.css` : styles personnalisés
